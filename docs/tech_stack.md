# Tech Stack for Recipe Book

The tech stack for Recipe Book is as follows:

### Frontend
 
* **Next.js**: Used for server-side rendering and static site generation.
* **Tailwind CSS**: Used for styling the application.
* **TypeScript**: Used for static type checking and code completion.

### Backend

* **Node.js**: Used as the runtime environment for the server.
* **Express**: Used as the web framework for building the API.
* **MongoDB**: Used as the database for storing recipes.
* **Redis**: Used for caching etc.

### Development Tools

* **ESLint**: Used for linting the codebase.
* **Prettier**: Used for formatting the codebase.
* **TypeScript**: Used for static type checking and code completion.
* **Husky**: Used for running linting and formatting scripts.

### Deployment

* **Vercel**: Used for deploying the application to a production environment.
* **render**: Used for deploying the backend.
